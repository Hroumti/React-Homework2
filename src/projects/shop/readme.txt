Shop Project - E-commerce Application

This project contains a complete e-commerce shop application with the following components:

- home.jsx: Main shopping page with navbar, carousel, search bar, filter, and product grid
- cart.jsx: Shopping cart page with product grid on left and order summary on right
- confirm.jsx: Order confirmation page with smaller product grid and payment form
- main.jsx: Main routing component that sets up all routes
- context.jsx: State management for cart functionality
- style.css: Complete styling for all components

Features:
- Responsive design that works on desktop, tablet, and mobile
- Interactive carousel with promotional slides
- Real-time product search and category filtering
- Shopping cart with add/remove/quantity controls
- Order confirmation with payment options (card/cash)
- Modern gradient design with smooth animations

To use this project, import main.jsx from this folder in src/App.jsx
